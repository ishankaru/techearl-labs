clickjacking-attacker

Attacker-controlled origin on port 8091. Hosts the PoC pages that frame the victim app at http://localhost:8090.

Reproduction

  1. Sign in at http://localhost:8090/login to set a session on the target origin.
  2. Open one of the attacker PoCs below in the same browser profile.
  3. Click as instructed. Return to the target dashboard and observe the “Your account was just deleted” banner.

PoCs

  • /classic.html — transparent iframe over a decoy button (classical clickjacking)
  • /cursor-jacking.html — fake cursor offset; the rendered cursor lies about the click position
  • /double-clickjacking.html — Paulos Yibelo 2024 variant using window.open + focus race (no iframe)

Defended reference

Each PoC also frames or opens /protected/dashboard for comparison. The browser refuses to render that route inside a frame because the response carries X-Frame-Options: DENY and Content-Security-Policy: frame-ancestors 'none'.
